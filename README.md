# E-Commerce Sales & Profitability Analysis

This project analyzes retail sales data to identify revenue trends, product profitability, regional performance, and the impact of discounts on business outcomes. The insights from this analysis are visualized through an interactive Power BI dashboard.

---

## Project Objectives

- Analyze sales performance across different product categories
- Identify the most profitable and loss-making product sub-categories
- Examine regional sales distribution
- Understand how discount levels impact profitability
- Build an interactive dashboard to support business decision making

---

## Tools & Technologies Used

- Python (Pandas, Matplotlib, Seaborn)
- SQL
- Power BI
- Jupyter Notebook

---

## Dataset

The dataset contains retail sales transactions including:

- Category
- Sub-Category
- Region
- State
- Segment
- Sales
- Profit
- Discount
- Quantity

Two dataset versions are included:

- **superstore_sales.csv** → original raw dataset  
- **cleaned_sales_data.csv** → cleaned dataset used for analysis and dashboard

---

## Key Analysis Performed

The project includes several analytical tasks:

- Sales performance analysis by product category
- Regional sales distribution analysis
- Profitability analysis by product sub-category
- Identification of loss-making product categories
- Customer segment sales analysis
- Discount vs profit impact analysis

## Key Insights

- Technology category generates the highest revenue.
- The West region contributes the largest share of sales.
- Certain sub-categories such as Tables generate negative profit.
- Higher discount levels are associated with reduced profitability.
